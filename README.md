# GroundnutAI

Static groundnut advisory chatbot powered by the Gemini API.

## GitHub Pages

1. Push `index.html`, `style.css`, and `app.js` to your GitHub repository root.
2. In GitHub, open `Settings` -> `Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose your main branch and `/root`.
5. Open the published site URL and add your Gemini API key in the Settings modal.

## Notes

- This project is frontend-only, so it works on GitHub Pages without a server.
- The Gemini API key is stored in the browser with `localStorage`.
- Each user should use their own Gemini API key.
